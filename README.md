# BALI TOUR 2026 - 旅のしおり Webサイト

体験型スライドショー形式の、データ駆動型デジタル旅のしおりです。

## ファイル構成

```
bali-tour-2026/
├── index.html        … ページの骨組み（通常は編集不要）
├── style.css          … デザイン・色・アニメーション
├── script.js          … 動作（データの流し込み・ナビゲーション等）
├── tour-data.json     … 旅行の内容（★ここを編集します）
└── assets/
    ├── images/        … 写真（ファイル名は tour-data.json と一致させる）
    ├── icons/         … アイコン画像（任意）
    ├── audio/         … BGM・効果音（mp3推奨）
    └── video/         … 動画（使用する場合）
```

## 内容を更新したいとき

日程・ホテル・グルメ・豆知識・FAQなどの文章は、すべて **`tour-data.json`** に
書かれています。HTML/CSS/JSを触らずに、このファイルの値を書き換えるだけで
サイトの表示が更新されます。

例：Day1の予定を変えたい場合は、`tour-data.json` 内の `itinerary` の
該当する `events` 配列を編集してください。

## 写真を追加したいとき

1. `tour-data.json` の中で写真ファイル名（例: `"gallery-sunset.jpg"`）を確認します。
2. Unsplash・Pexelsなどのフリー素材サイトから写真をダウンロードします。
3. 同じファイル名にリネームして `assets/images/` フォルダに入れます。

写真がまだ無い場所は、「ここに写真を追加してください」という案内が
自動で表示されるので、サイトが崩れることはありません。

### 写真クレジットについて

現在 `assets/images/` に入っている写真は、Wikimedia Commonsで公開されている
オープンライセンス（CC-BY / CC-BY-SA など）の写真を使用しています。
身内・グループ内で共有する範囲であれば問題ありませんが、もし今後
不特定多数に公開する（SNSでの二次利用など）場合は、念のため
各画像のWikimedia Commonsページでライセンス表記・クレジット表記の要否を
ご確認ください。ご自身で撮影した写真やUnsplash/Pexelsの写真に差し替える場合は
特に気にする必要はありません。

## BGM・効果音について

`assets/audio/` フォルダには、以下の音声を入れています。

- `waves.wav` … 波の音（常時ループ／自動生成音）
- `gamelan.mp3` … メインBGM（ご提供いただいた音源を使用・常時ループ）
- `sfx-plane.wav` … 飛行機が飛び立つ効果音（オープニング演出用・自動生成音）

BGMは、タイトルロゴが表示された時点で自動再生を試みます。ブラウザの
自動再生制限で最初はブロックされることがありますが、その場合は画面を
1回タップ・クリックすると再生が始まります。

`gamelan.mp3` のみ、いただいた実際の音源に差し替え済みです。他の音を
差し替えたい場合も、同じファイル名で上書きすれば反映されます。

### BGMを選べる機能について

画面右上のBGMボタン（🔊）をタップすると、曲を選べるメニューが開きます。
選択肢は `tour-data.json` の `bgmTracks` で管理しているので、新しい曲を
追加したいときは、音声ファイルを `assets/audio/` に入れたうえで、
`bgmTracks` に曲名とファイル名を追記してください（AIに頼めば代わりに
追加できます）。

### もっと本格的な音に差し替えたいとき

Pixabay Music・YouTubeオーディオライブラリなどの著作権フリー音源サイトから
音声ファイルをダウンロードし、同じファイル名（例: `waves.wav` や `waves.mp3`
などにリネーム）で `assets/audio/` フォルダに上書きしてください。
mp3・wavのどちらの形式でも再生できます（ファイル名の拡張子を変える場合は、
`index.html` 内の `<audio src="...">` の部分も同じ拡張子に書き換えてください）。

## ローカルで確認する方法（重要）

`index.html` をブラウザにドラッグ＆ドロップして直接開くと、ブラウザの
セキュリティ制限により `tour-data.json` を読み込めないことがあります
（読み込み画面から進まない場合はこれが原因です）。

その場合は、簡易サーバー経由で開いてください。

- VSCodeを使っている場合：「Live Server」拡張機能をインストールし、
  `index.html` を右クリック →「Open with Live Server」
- コマンドラインの場合：このフォルダで `python -m http.server 8000` を実行し、
  ブラウザで `http://localhost:8000` を開く

## 通貨換算のコツ（Chapter 6 内で紹介）

ルピア（Rp）は桁が大きいため、「0（ゼロ）を3つ消す」と日本円の感覚に近づきます。
例：100,000Rp → 0を3つ消して 100 → だいたい100円台〜1,000円台のイメージ
（正確なレートは `tour-data.json` の `currency.rateJpyToIdr` で調整できます）。
